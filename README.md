# Happy Apps

Happy Apps — a personal blog by Joel Sooriah, built with Toucan, a server-side Swift static site generator.                                

Stack                                                                                                                                    
- Generator: Toucan (Swift-based, markdown-first, convention-over-config)                                                                
- Theme: Custom default theme using Mustache templates + CSS                                                                             
- Deployment: GitHub Pages (jsooriah.github.io/happyapps)                                                                                
- Build: make dev/dist/watch/serve commands wrapping toucan generate                                                                       

Content Focus                                                                                                                              

The blog sits at the intersection of product management and AI engineering, written from Joel's perspective as someone who transitioned    
from engineering into PM. Core themes:

- Product thinking — JTBD framework, root cause analysis, customer-centric design                                                          
- AI/agents — agent memory landscape, agentic RAG tools, UX in the AI era
- Team dynamics — remote teams, cross-functional alignment, Agile roots                                                                    
- Personal essays — human-centered design in an over-digitized world                                                                       
  
Notable Posts                                                                                                   
- The Agent Memory Landscape — PM guide to context-aware AI systems                                                                      
- Building an Agentic RAG Tool — for researching French Revolution history in the Indian Ocean
- Gherkin Syntax as Rosetta Stone — cross-functional alignment via JTBD                                                                  
- The Silent Kitchen (Sébastien Bras) — product & team building lessons from a chef                                                        
  
Tooling                          
- research.py — a Python script that pulls from a local Lenny's Podcast/Newsletter dataset to generate a "Voices from the Field" section, calling Claude's API to enrich blog posts with curated references.                                                                                                 
  A personal, thoughtfully crafted blog that doubles as an AI engineering sandbox.
